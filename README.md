# 머니 로그 (Money Log)

가계부 Android 앱

## 프로젝트 정보

- **패키지명**: com.moneylogs.app
- **최소 SDK**: API 26 (Android 8.0)
- **타겟 SDK**: API 36
- **버전**: 1.6 (versionCode 7)
- **언어**: Kotlin

## 개발 환경

- Android Studio
- Kotlin
- Gradle (Kotlin DSL)

## 빌드 방법

```bash
./gradlew assembleDebug
```

## 주요 의존성

- AndroidX Core KTX
- AndroidX AppCompat
- Material Design
- SplashScreen
- ConstraintLayout
- In-App Update (Play Core)
